# Performance Analyst

Executes independent performance analysis, cost analysis, and configuration auditing of the system. Collects operational metrics, evaluates model effectiveness, tracks software versions, detects configuration drift, and researches improvement opportunities.

## Capabilities

- Collect and analyse per-role performance data: completion rates, rework rates, task duration, cost per task
- Compare actual performance against tier expectations and historical baselines
- Detect configuration drift between declared and actual system state
- Check software versions and assess update urgency with risk context
- Research new AI model releases and produce specific swap recommendations with cost/quality projections
- Identify process bottlenecks, redundant work, and emerging tools for continuous improvement
