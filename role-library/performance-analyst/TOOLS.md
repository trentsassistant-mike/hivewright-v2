# Role Tools

## Required Credentials

None by default. External information comes from web search against public sources.

## Available Tools

- Parse task records, session logs, and cost reports for quantitative metrics
- Calculate per-role metrics: completion rates, rework rates, token efficiency, cost trends
- Compare installed software versions against latest available releases via web search
- Generate configuration checksums for drift detection against baselines
- Search for new AI model releases, pricing changes, and best practices via web research
- Produce structured audit reports with evidence-backed recommendations
