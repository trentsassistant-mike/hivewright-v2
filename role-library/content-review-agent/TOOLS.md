# Role Tools

## Required Credentials

None by default.

## Available Tools

- Web search for verifying claims, sources, statistics, regulations, and competitive statements
- Web fetch for reading cited sources in full when search snippets are insufficient
- Python and shell calculators (bc) for arithmetic verification in financial deliverables
- JSON parsing tools for reviewing structured outputs
- Shell utilities for verifying referenced file paths exist and documented commands work
- Text search tools for searching deliverables and supporting files
