# Role Tools

## Required Credentials

None by default.

## Available Tools

- Web search for discovering market data, government statistics, industry reports, and financial databases
- Web fetch for reading full pages of authoritative sources such as government portals and company filings
- Browser for interactive data portals and JavaScript-rendered statistics sites
- Python for market sizing calculations, growth projections, and statistical computations
- Image analysis for extracting data points from charts, graphs, and infographics
- Shell calculator tools (bc, date) for arithmetic verification and date calculations
