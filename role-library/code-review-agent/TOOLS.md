# Role Tools

## Required Credentials

Git and GitHub credentials configured on the host for git and gh commands.

## Available Tools

- Static analysis tools: Semgrep, Ruff, ESLint, mypy, tsc for code quality and type checking
- Security scanners: Trivy, Gitleaks, Bandit, npm audit, pip-audit, Checkov for vulnerability and secret detection
- Hallucination verification commands for import checking, dependency existence, build verification, and test execution
- Git tools for reviewing change history and comparing file versions
- Shell utilities for codebase search (ripgrep), line counts, and file comparison
- GitHub MCP for CI status, PR context, and Dependabot alerts
