# Role Tools

## Required Credentials

None by default. Web search may be used when the task brief explicitly requires vendor or policy discovery.

## Available Tools

- Read and write operational workflow artifacts, vendor records, and people-ops records
- Build comparison matrices and procurement packets from gathered inputs
- Search the web for vendor pricing, product specifications, or policy references when briefed
- Parse structured data for comparisons and date calculations
- Generate completion reports detailing outcomes, updated records, and outstanding blockers
