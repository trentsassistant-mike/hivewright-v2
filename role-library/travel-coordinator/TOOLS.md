# Role Tools

## Required Credentials

None by default. Travel platform APIs (Amadeus MCP, Google Maps MCP) added via skills when available.

## Available Tools

- Web search for flight options, hotel comparisons, visa requirements, travel advisories, and destination research
- Web fetch for airline policies, hotel amenities, visa procedures, and government travel advice
- Timezone arithmetic and date calculation utilities for itinerary planning
- Inline calculations via Python for duration computation and price comparison
