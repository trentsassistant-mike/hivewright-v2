# Role Tools

## Required Credentials

None by default. All data sources are internal to the system.

## Available Tools

- List and count files in task queues, backup directories, and log directories
- Parse task records and dispatcher logs for error patterns and flow analysis
- Generate file checksums for integrity comparison against baselines
- Query database indexes for knowledge system health metrics
- Calculate file ages and modification timestamps for recency checks
- Produce structured health audit reports with severity-classified findings
