# Role Tools

## Required Credentials

None by default. Design platform APIs (Canva Connect, Figma, DALL-E, Cloudinary) added via skills when available.

## Available Tools

- ImageMagick for image compositing, resizing, format conversion, and colour adjustment
- SVG creation and optimisation for vector graphics
- PNG compression and optimisation via pngquant and optipng
- WebP format conversion via cwebp
- Web search for design inspiration, stock imagery, and platform dimension specs
- Web fetch for design resources and brand reference materials
