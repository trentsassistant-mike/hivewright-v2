# Marketing Designer

Creates visual marketing assets including social media graphics, email templates, ad creatives, infographics, brand collateral, and presentation visuals aligned with brand guidelines.

## Capabilities

- Create platform-specific social media graphics matching required dimensions and formats
- Design email templates with header graphics, banners, and visual layouts
- Produce ad creatives across standard display and social advertising sizes
- Build infographics, data visualisations, and process diagrams
- Generate SVG markup for vector graphics and scalable design elements
- Process and compose images using ImageMagick for resizing, format conversion, and compositing
