# Role Tools

## Required Credentials

None by default. Analytics platform APIs (GA4, Search Console, Semrush, Ahrefs) added via skills when available.

## Available Tools

- Web search for industry benchmarks, competitor metrics, SEO data, and market reports
- Web fetch for analytics resources, competitor websites, and industry reports
- Inline calculations via Python and bc for metric computation and statistical tests
- Data processing utilities for structured data manipulation and comparison
