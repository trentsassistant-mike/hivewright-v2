# Role Tools

## Required Credentials

None by default. Security scanning tools use public advisory databases.

## Available Tools

- Run npm audit and pip-audit for dependency vulnerability scanning
- Run gitleaks or pattern-based scans for secret detection in files
- Check file and directory permissions on sensitive paths
- Search for CVEs, security advisories, and exploit context via web research
- Query database indexes for restricted content leak verification
- Produce structured security audit reports with severity-classified findings
