# Role Tools

## Required Credentials

- Xero API (read-only access for pulling financial report data)

## Available Tools

- Pull financial reports from Xero API (P&L, balance sheet, transactions, invoices)
- Run cost report aggregation for operational spend analysis
- Perform financial calculations, forecasting models, and data processing
- Search for and verify current API pricing, exchange rates, tax rates, and financial benchmarks
- Parse structured financial and cost data in JSON format
- Access provider billing dashboards for cost reconciliation when needed
