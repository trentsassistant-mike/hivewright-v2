# Role Tools

## Required Credentials

None by default. CRM platform credentials when CRM integration is available.

## Available Tools

- Web search for prospect research, competitive intelligence, market analysis, and company information
- Web fetch for retrieving company websites, press releases, job postings, and pricing pages
- Image analysis for reviewing marketing collateral and visual materials referenced in briefs
- Python and shell calculators for pricing calculations, ROI projections, and discount computation
- Date utilities for deadline and follow-up calculations
- JSON parsing tools for pipeline data and deal record management
