# Role Tools

## Required Credentials

None by default. Legal citation verification is conducted via web search. Specialised legal databases are escalated as tool gaps when needed.

## Available Tools

- Read existing contracts, templates, clause libraries, and department standards
- Write contract drafts, review reports, redlines, and employment document packages
- Search for and verify current legislation, statutory references, and case law affecting contract terms
- Fetch authoritative legal source pages for jurisdiction-specific requirements
- Compare contract versions for redline production
- Perform deadline and notice period calculations
