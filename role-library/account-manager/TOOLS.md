# Role Tools

## Required Credentials

None by default. CRM platform credentials when CRM integration is available.

## Available Tools

- Web search for customer research, industry benchmarks, and satisfaction best practices
- Web fetch for customer company updates, press releases, and news for proactive relationship management
- Image analysis for reviewing customer-provided documents and screenshots
- Python and shell calculators for satisfaction score calculations, renewal pricing, and account metrics
- Date utilities for renewal reminder and deadline calculations
- JSON parsing tools for customer data and account record management
