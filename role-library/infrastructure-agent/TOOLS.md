# Role Tools

## Required Credentials

- Platform CLI credentials (Railway, AWS, GCP, Vercel, Fly.io as needed)
- Git and GitHub credentials for branch verification and deployment

## Available Tools

- Run platform CLIs for deployment and environment management
- Run IaC tools (Terraform, Tofu) for infrastructure provisioning
- Run Docker operations for container builds and local stack testing
- Run secret scanning tools (gitleaks, trivy) before commits and deployments
- Perform health check verification post-deployment via HTTP requests
- Read and write environment runbooks and infrastructure change logs
