# Scheduler

Executes time-based coordination work: meetings, booking windows, recurring cadences, deadline plans, interview schedules, onboarding calendars, review timelines, and other tasks where the primary constraint is time or resource availability.

## Capabilities

- Build and update schedules for meetings, workflows, review cycles, and recurring cadences
- Resolve time and resource conflicts using constraints provided in the task brief
- Produce scheduling artifacts: run sheets, booking schedules, timeline tables, deadline sequences
- Coordinate interview blocks, onboarding timetables, and renewal deadline plans
- Track critical dates and flag when a requested timetable is infeasible
- Perform date arithmetic and constraint-based schedule generation
