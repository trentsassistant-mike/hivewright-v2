# Role Tools

## Required Credentials

None by default.

## Available Tools

- Web search for competitor information, industry news, social media discussions, trend data, and job postings
- Web fetch for reading competitor product pages, news articles, press releases, and industry publications
- Browser for Google Trends analysis, social media search interfaces, and JavaScript-rendered competitor sites
- Image analysis for competitor screenshots, marketing materials, and product UI comparisons
- Python for data structuring and comparison matrix generation
- Shell utilities for JSON parsing and content measurement
