# Role Tools

## Required Credentials

- Figma API key (optional, for extracting design context from existing Figma files)

## Available Tools

- Read and analyse existing specifications, design systems, and project codebases
- Write product specs, component specs, design system docs, and audit reports
- Run accessibility CLI tools (Pa11y, axe-core) for headless WCAG audits
- Validate design token files against W3C DTCG specification
- Analyse UI screenshots, mockups, and wireframes
- Search for UX patterns, WCAG guidance, and accessibility references
