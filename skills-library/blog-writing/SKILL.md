# Blog Writing

## When to use
When creating blog posts, articles, or long-form content for a hive website.

## Procedure
1. Research the topic using provided keywords and hive context
2. Create an outline with H2/H3 structure
3. Write the first draft (target 800-1200 words)
4. Include internal links to existing content where relevant
5. Add meta description (under 160 characters)
6. Write a compelling title (under 60 characters)

## Quality checklist
- [ ] Keyword naturally appears in title, first paragraph, and at least 2 H2s
- [ ] No paragraphs longer than 3 sentences
- [ ] Includes at least one call-to-action
- [ ] Spell-checked and grammar-checked
